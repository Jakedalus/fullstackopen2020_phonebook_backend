# phonebook_backend

> The backend portion of Full Stack Open's Part 3

Full Stack Open 2020 can be found here https://fullstackopen.com/

## Usage

```js
var phonebookBackend = require('phonebook_backend')

```

```

$ npm install phonebook_backend
```
